Programming Languages

프로그래밍 언어는 표현 방법 (문법) 과 의미 (의미론) 를 규정하며, 하드웨어 독립성과 추상화 레벨에 따라 고/저급 언어로 분류된다. 컴파일러·인터프리터·JIT 기반의 구현 방식과 객체지향, 함수형, 절차적 같은 패러다임을 통해 다양한 문제 해결에 최적화돼 있으며, 실무에서는 성능, 안전성, 유지보수성을 고려해 선택된다.
