Virtual Box#

가상화 소프트웨어로, 물리적 컴퓨터에서 여러 가상 머신(VM)을 실행할 수 있게 해주는 도구.

  1. 회사 및 라이선스:

    • Oracle Corporation에서 개발
    • 오픈 소스 소프트웨어로, 기본 버전은 무료로 사용 가능

  2. 성능 및 기능:

    • VMware에 비해 성능이 다소 떨어질 수 있음
    • 기본적인 가상화 기능(스냅샷, 클론, 3D 가속 등) 제공
    • 일부 고급 기능은 확장 팩 설치 필요

  3. 호환성:

    • Windows, macOS, Linux, Solaris 등 다양한 호스트 및 게스트 OS 지원

  4. 사용자 인터페이스:

    • 비교적 간단한 인터페이스 제공, 일부 기능은 탐색이 어려울 수 있음

  5. 지원:

    • 활발한 오픈 소스 커뮤니티를 통한 지원
