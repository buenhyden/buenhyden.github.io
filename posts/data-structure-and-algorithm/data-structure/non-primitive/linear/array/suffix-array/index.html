Suffix Array#

Suffix Array는 주어진 문자열의 모든 접미사(suffix)를 사전순으로 정렬한 배열로, 각 요소는 접미사의 시작 위치를 나타내는 정수이다.

특징#

  1. 모든 접미사를 사전순으로 정렬하여 저장한다.
  2. 공간 효율적인 구조를 가진다.
  3. 빠른 문자열 검색을 가능하게 한다.

장점#

  1. 효율적인 문자열 검색과 패턴 매칭이 가능하다.
  2. 공간 복잡도가 O(n)으로 효율적이다.
  3. 다양한 문자열 관련 문제 해결에 활용될 수 있다.

단점#

  1. 구축 과정이 복잡할 수 있다.
  2. 기본 구현의 시간 복잡도가 O(n^2 log n)으로 높다.

응용#

  1. 문자열 검색 및 패턴 매칭
  2. 생물정보학에서의 DNA 시퀀싱 분석
  3. 데이터 압축 알고리즘
  4. 텍스트 인덱싱 및 전체 텍스트 검색

동작 원리#

  1. 문자열의 모든 접미사를 생성한다.
  2. 생성된 접미사들을 사전순으로 정렬한다.
  3. 정렬된 접미사의 시작 위치를 배열에 저장한다.

구성 요소#

  1. 원본 문자열
  2. 정수 배열 (접미사의 시작 위치를 저장)
  3. (선택적으로) LCP(Longest Common Prefix) 배열

구현 방식#

일반적으로 다음과 같은 방식으로 구현된다:

  1. 나이브한 방법: O(n^2 log n) 시간 복잡도
  2. 맨버-마이어스 알고리즘: O(n log^2 n) 시간 복잡도
  3. SA-IS 알고리즘: O(n) 시간 복잡도 (고급 구현)

주요 연산들의 동작 과정#

  1. 구축 (Build):

    • 모든 접미사를 생성하고 정렬한다.
    • 정렬된 접미사의 시작 위치를 배열에 저장한다.

  2. 검색 (Search):

    • 이진 검색을 사용하여 패턴을 찾는다.

  3. LCP 계산:

    • 인접한 접미사들 간의 최장 공통 접두사 길이를 계산한다.

class SuffixArray:
    def __init__(self, text):
        self.text = text
        self.suffixes = sorted(range(len(text)), key=lambda i: text[i:])

    def search(self, pattern):
        left, right = 0, len(self.suffixes)
        while left < right:
            mid = (left + right) // 2
            suffix = self.text[self.suffixes[mid]:]
            if pattern > suffix:
                left = mid + 1
            elif pattern < suffix:
                right = mid
            else:
                return self.suffixes[mid]
        return -1

# 사용 예
sa = SuffixArray("banana")
print(sa.search("ana"))  # 출력: 1
