이진 검색 (Binary Search)#

이진 검색은 정렬된 리스트에서 특정 값을 찾는 효율적인 알고리즘이다.
이 알고리즘은 리스트의 중간 값을 선택하고, 찾고자 하는 값과 비교하여 탐색 범위를 반으로 줄여가며 검색을 수행한다.

장점#

  1. 검색 속도가 매우 빠릅니다. 시간 복잡도는 O(log n)입니다.
  2. 대용량 데이터에서 특정 값의 위치를 찾는 데 효율적입니다.

단점#

  1. 반드시 정렬된 데이터에서만 사용할 수 있습니다.
  2. 데이터의 삽입, 삭제가 빈번한 경우 비효율적일 수 있습니다.

주의해야 할 점들#

  1. 정렬 상태 유지:

    • 데이터 삽입/삭제 시 정렬 상태를 유지해야 합니다.
    • 데이터 변경이 빈번한 경우 다른 자료구조를 고려해야 합니다.

  2. 중간 인덱스 계산:

    • 오버플로우를 방지하기 위해 mid = left + (right - left) / 2 형태를 사용합니다.
    • 단순히 (left + right) / 2를 사용하면 큰 숫자에서 문제가 발생할 수 있습니다.

  3. 경계 조건 처리:

    • 배열이 비어있는 경우
    • 찾는 값이 배열의 범위를 벗어나는 경우
    • 중복된 값이 있는 경우

이진 검색의 실제 활용 사례#

  1. 데이터베이스 인덱싱:

    • B-트리나 B+트리의 기본 원리로 사용됩니다.
    • 대용량 데이터베이스의 빠른 검색을 가능하게 합니다.

  2. 버전 관리:

    • 버그가 처음 발생한 버전을 찾을 때 사용됩니다.
    • git bisect 명령어가 이진 검색을 활용합니다.

  3. 시스템 설계:

    • 로드 밸런서의 용량 계획
    • 캐시 크기 최적화

  4. 수치 해석:

    • 방정식의 근사해 찾기
    • 최적화 문제 해결

이진 검색을 효과적으로 활용하기 위한 팁#

  1. 데이터 정렬 비용 고려:

    • 한 번 정렬하고 여러 번 검색하는 경우에 효율적입니다.
    • 정렬 비용이 검색 이점을 상쇄하지 않는지 검토해야 합니다.

  2. 변형 알고리즘 활용:

    • Lower bound / Upper bound 검색
    • 가장 가까운 값 찾기
    • 순환 정렬된 배열에서의 검색

  3. 최적화 기법:

    • 캐시 지역성 활용
    • 분기 예측 최적화
    • SIMD 명령어 활용 (하드웨어 가속)

주로 사용하는 데이터 구조#

이진 검색은 주로 배열이나 이진 탐색 트리와 같은 정렬된 선형 자료구조에서 사용된다.

이러한 데이터 구조를 사용하는 이유:

  1. 배열: 연속된 메모리 공간에 데이터를 저장하여 인덱스를 통한 빠른 접근이 가능합니다.
  2. 이진 탐색 트리: 트리 구조를 통해 효율적인 삽입, 삭제, 검색 연산이 가능합니다.

// Java Implementation
public class BinarySearch {
    // 기본 이진 검색 구현
    public static int search(int[] arr, int target) {
        int left = 0;
        int right = arr.length - 1;
        
        while (left <= right) {
            // 중간 인덱스 계산 (오버플로우 방지를 위한 방식)
            int mid = left + (right - left) / 2;
            
            // 중간 값과 타겟 비교
            if (arr[mid] == target) {
                return mid;  // 타겟을 찾은 경우
            } else if (arr[mid] < target) {
                left = mid + 1;  // 오른쪽 절반 탐색
            } else {
                right = mid - 1;  // 왼쪽 절반 탐색
            }
        }
        return -1;  // 타겟을 찾지 못한 경우
    }
    
    // 가장 왼쪽에 있는 타겟 값의 인덱스를 찾는 구현
    public static int searchLeftmost(int[] arr, int target) {
        int left = 0;
        int right = arr.length - 1;
        int result = -1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            
            if (arr[mid] == target) {
                result = mid;  // 현재 위치 저장
                right = mid - 1;  // 왼쪽도 계속 탐색
            } else if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return result;
    }
    
    // 재귀적 구현
    public static int searchRecursive(int[] arr, int target, int left, int right) {
        if (left > right) {
            return -1;
        }
        
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target) {
            return mid;
        } else if (arr[mid] < target) {
            return searchRecursive(arr, target, mid + 1, right);
        } else {
            return searchRecursive(arr, target, left, mid - 1);
        }
    }
}

// JavaScript Implementation
class BinarySearch {
    // 기본 이진 검색
    static search(arr, target) {
        let left = 0;
        let right = arr.length - 1;
        
        while (left <= right) {
            const mid = Math.floor(left + (right - left) / 2);
            
            if (arr[mid] === target) {
                return mid;
            } else if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return -1;
    }
    
    // 범위 내의 값 개수 찾기
    static countInRange(arr, lowerBound, upperBound) {
        const leftIndex = this.findLowerBound(arr, lowerBound);
        const rightIndex = this.findUpperBound(arr, upperBound);
        
        if (leftIndex === -1 || rightIndex === -1) {
            return 0;
        }
        return rightIndex - leftIndex + 1;
    }
    
    // Lower bound 찾기
    static findLowerBound(arr, target) {
        let left = 0;
        let right = arr.length - 1;
        let result = -1;
        
        while (left <= right) {
            const mid = Math.floor(left + (right - left) / 2);
            
            if (arr[mid] >= target) {
                result = mid;
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return result;
    }
}

# Python Implementation
class BinarySearch:
    @staticmethod
    def search(arr: list, target: int) -> int:
        """기본 이진 검색"""
        left, right = 0, len(arr) - 1
        
        while left <= right:
            mid = (left + right) // 2
            
            if arr[mid] == target:
                return mid
            elif arr[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        
        return -1
    
    @staticmethod
    def search_closest(arr: list, target: int) -> int:
        """가장 가까운 값 찾기"""
        if not arr:
            return -1
            
        left, right = 0, len(arr) - 1
        
        while left + 1 < right:
            mid = (left + right) // 2
            
            if arr[mid] == target:
                return mid
            elif arr[mid] < target:
                left = mid
            else:
                right = mid
        
        # 가장 가까운 값 결정
        if abs(arr[left] - target) <= abs(arr[right] - target):
            return left
        return right
    
    @staticmethod
    def search_insert_position(arr: list, target: int) -> int:
        """삽입 위치 찾기 (정렬된 상태 유지)"""
        left, right = 0, len(arr)
        
        while left < right:
            mid = (left + right) // 2
            
            if arr[mid] < target:
                left = mid + 1
            else:
                right = mid
        
        return left
