프론트엔드 기초(Frontend Fundamentals)

프론트엔드 개발의 핵심 기술인 HTML, CSS, JavaScript의 기본 개념과 원리를 다룹니다. 웹 페이지의 구조를 정의하는 HTML, 시각적 스타일을 담당하는 CSS, 동적 기능을 구현하는 JavaScript의 기초와 함께 웹 API의 활용법까지 프론트엔드 개발의 토대가 되는 지식을 제공합니다.

HTML
