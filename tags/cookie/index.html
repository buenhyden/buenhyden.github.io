Cookie Based Auth

Cookie Based Auth 웹 애플리케이션에서 사용자 세션을 유지하는 전통적인 방법 사용자의 인증 정보를 클라이언트 측의 쿠키에 저장하여 인증 상태를 유지하는 방식으로 Stateless한 HTTP 연결을 Stateful하게 만든다. 주요 특징 보안적 특징: 쿠키는 HttpOnly 플래그로 JavaScript 접근을 방지할 수 있습니다.
