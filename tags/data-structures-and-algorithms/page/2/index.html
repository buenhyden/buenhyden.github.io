Recursion vs. Iteration

Iteration은 명시적인 반복 구조를 가지며, 각 반복마다 변수의 상태가 변경된다. Recursion은 함수가 자기 자신을 호출하여 문제를 해결하는 방식이다. 복잡한 문제를 더 작고 간단한 문제로 나누어 해결한다. Recursion은 base case(종료 조건)와 recursive case(재귀 호출)로 구성된다.
