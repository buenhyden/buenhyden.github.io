ECMAScript

ECMAScript ECMAScript는 자바스크립트의 표준 규격으로, Ecma International이 ECMA-262 기술 규격을 통해 정의한 스크립트 프로그래밍 언어이다. 1997년 처음 표준화되었으며, 자바스크립트의 핵심 문법과 기능을 규정한다. Hoisting
